## 🗺️ 一、全文逻辑结构

```
引子：夏天是"双头怪"——既有玩乐，也有热浪（heat wave）；高温影响一切，包括饮食
  │
  ├─ 高温会抑制食欲（appetite-suppressant）：身体为散热(thermoregulate)、下丘脑热感蛋白制造"饱腹感"
  │     └─ 但营养需求不变 → 关键是选对能补「水分/电解质/营养」的食物
  │
  ├─ ❌ 不该吃/喝：
  │     ├─ 油腻厚重(heavy, greasy)：难消化、升体温、让人发懒(sluggish)
  │     ├─ 高盐加工食品：增钠 → 加重脱水(dehydration)
  │     ├─ 冷啤酒 & 冰茶：酒精/咖啡因是利尿剂(diuretics) → 排水
  │     └─ 全脂奶(减慢胃排空、升温) & 高糖饮料(把水分拉进肠道)
  │
  ├─ ✅ 该吃：
  │     ├─ 清淡冷蛋白：水煮蛋、金枪鱼/冷虾沙拉、豆类沙拉
  │     └─ ⚠️ 易腐(perishable)！注意食品安全（冷藏<40°F，室温别放超1小时）
  │
  ├─ 💧 喝比吃更重要：
  │     ├─ 全天小口补水(sip)、看尿色判断是否脱水
  │     └─ 不只补水，还要补电解质(electrolytes)：运动饮料、椰子水、酸樱桃汁
  │
  ├─ 🍉 果蔬 = 水分+电解质的富源：果蔬平均含水80%，西瓜90%，黄瓜97%
  │     ├─ 冰沙(smoothies)：既补水又不像奶昔那样高糖高脂
  │     └─ 辣食：刺激出汗→最终降温（但先补足水，以免刺激肠胃）
  │
  └─ 结论：热浪饮食 = 补水 + 明智营养选择的平衡；"好吃"和"健康"可以兼得
```

**一句话主旨自测**——

<details><summary>▶ 参考答案</summary>

热浪天高温会抑制食欲但营养需求不减；应避开油腻、高盐、酒精咖啡因、全脂奶和高糖，多吃清淡冷蛋白和高含水果蔬，重点是补水和电解质——只要平衡好补水与营养，就能既吃得开心又吃得健康。

</details>

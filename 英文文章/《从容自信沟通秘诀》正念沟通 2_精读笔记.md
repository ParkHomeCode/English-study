## 🗺️ 一、全文逻辑结构

```
主题：正念沟通 = 少关注"我说什么"，多关注"如何倾听别人"
  │
  ├─ 倾听为何难？三大障碍，都以 P 开头：
  │     ├─ Physical 物理的（环境噪音）
  │     ├─ Physiological 生理的（累/饿/紧张）
  │     └─ Psychological 心理的（忙着排练要说的话、评判、联想）
  │
  ├─ 倾听是「技能」，可后天习得，不是天生（not innate）
  │
  ├─ 怎么听得更好？口诀 ACE = Space · Pace · Grace（押韵）
  │     ├─ Space 空间：物理安静处 + 心理留白（活在当下）
  │     ├─ Pace 节奏：慢下来、专注
  │     └─ Grace 余地：允许自己"听到言外之意"
  │
  ├─ 反面故事：同事要的是"支持"，他却急着"给反馈" → 没真正听懂
  │
  ├─ 战术：像"准备复述"那样去听（listen to paraphrase）
  │     └─ 听 bottom line（真正重点）→ 复述 → 显示在乎 → 建立信任（关系的基石）
  │
  └─ 练习法：每次互动后自问"那是关于什么？我能怎么用？"
        └─ 内在复述 → 训练大脑神经通路，让复述能力越来越强
```

**一句话主旨自测**——

<details><summary>▶ 参考答案</summary>

好的沟通重在**倾听**而非表达；倾听有三大障碍（物理/生理/心理），但它是可习得的技能。靠 **Space-Pace-Grace** 和"**复述式倾听**"听到对方的真正心声、显示在乎，就能建立信任——而信任是一切关系的基础。

</details>
